$f : \mathbb{R} → \mathbb{R}$ definida por $f(x) = [x]$ se conoce con el nombre de función mayor entero. $[x]$ es el mayor entero menor o igual a $x$ así $[2.3] = 2$, $[−4.37] = −5$, $[1] = 1$, etc. Para esta función $D_f = \mathbb{R}, \mathbb{R}_f = Z$.
