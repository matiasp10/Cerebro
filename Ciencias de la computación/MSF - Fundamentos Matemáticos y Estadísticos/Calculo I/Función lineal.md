Sea $f : \mathbb{R} → \mathbb{R}$ definida por $f(x) = ax + b$, con $a, b ∈ \mathbb{R}$, se llama **función lineal** y para esta función $D_f = C_f = \mathbb{R}_f = \mathbb{R}$. La gráfica de $f(x) = 3x + 1$, es:
